# RGB + Event Camera System Wiki

Welcome! This wiki is a practical guide for setting up and using the RGB + stereo event camera system on Ubuntu 20.04 + ROS Noetic. It is written for newcomers and industry users who want a clear, reliable workflow.

## Quick Start

1) Follow the setup guide: `Setup.md`
2) Run calibration: `Calibration.md`
3) Record data and extract frames: `Data-Extraction.md`
4) Understand the dataset format: `Dataset-Format.md`
5) Run annotation tools: `Annotation.md`
